# Weather App

A modern weather web application built with JavaScript that provides real-time weather information for any city. The app fetches data from the OpenWeatherMap API and displays it in an interactive and user-friendly interface.

## Features

- Search for any city and get current weather data.
- Displays:
  - Current temperature
  - Minimum and maximum temperature
  - Weather condition description
  - Humidity
  - Wind speed
  - Weather icon based on the condition
- 5-day forecast showing maximum and minimum temperatures.
- Clickable forecast items to open a modal with detailed daily information:
  - Date
  - Temperature (high/low)
  - Humidity
  - Wind speed
  - Weather condition
- Dynamic date and time adjusted to the searched city.
- Smooth animations for forecast items and modal.

## Technologies

- HTML5 & CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- DOM manipulation and event handling
- Flexbox layout
- CSS transitions and animations
